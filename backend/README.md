# RehabCare Backend

Это серверная заготовка под реальную интеграцию:

- `Redis` для сессий, очередей уведомлений, хранения живых чатов и presence
- `Express` как стартовая HTTP-обвязка

Пока здесь только минимальный health-check и конфиг подключения к Redis.

Следующий практический шаг:

1. хранение `chatThreads` и `incidents` в Redis/PostgreSQL
2. websocket-канал для чата врача и пациента
3. auth для врача и пациента
4. API для дневника самоконтроля, задач, ЛФК и уведомлений
