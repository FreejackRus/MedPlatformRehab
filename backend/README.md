# Серверная часть MedPlatformRehab

Сервер реализует:

- авторизацию пациента и врача;
- выдачу рабочих данных для кабинета пациента и врача;
- сохранение дневника самоконтроля, задач, упражнений и напоминаний;
- чат пациента и врача;
- WebSocket-канал для обновления переписки;
- подключение к Redis с автоматическим переходом на in-memory режим, если Redis недоступен.

## Запуск локально

```bash
cd backend
npm install
npm run dev
```

Сервер по умолчанию поднимается на `http://localhost:4010`.

## Переменные окружения

- `PORT` — порт backend-сервиса, по умолчанию `4010`
- `REDIS_URL` — адрес Redis, по умолчанию `redis://127.0.0.1:6379`

## Проверка состояния

```bash
curl http://localhost:4010/health
```

## Docker

Backend можно запускать как отдельный контейнер или вместе с frontend и Redis через `docker-compose.yml` в корне проекта.
